Animation Properties

1. animation-name Property

The animation will not run visibly unless a duration is specified

2. animation-duration Property

animation-duration: 2s; The animation completes in 2 seconds. The animation now runs visibly after a duration is specified

3. animation-timing-function Property

animation-timing-function: ease-in-out; The animation: Starts slowly Speeds up in the middle Slows down at the end

3. animation-delay Property

The animation-delay property sets a waiting time before the animation starts. animation-delay: 1s; The animation waits 1 second before starting.
