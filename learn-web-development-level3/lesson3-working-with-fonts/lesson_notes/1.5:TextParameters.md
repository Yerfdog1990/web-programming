# Text Parameters in CSS

Text parameters control the spacing and readability of text on web pages. The most important properties are:

* `line-height` → Controls space between lines
* `letter-spacing` → Controls space between characters
* `word-spacing` → Controls space between words

These properties improve readability, visual balance, and design aesthetics.

---

# 5.1 The `line-height` Property

The `line-height` property defines the vertical space between lines of text. It controls how dense or spaced text appears.

## Why It Matters

* ✔ Improves readability
* ✔ Prevents text from looking crowded
* ✔ Creates comfortable reading experience

---

## Values of `line-height`

| Value Type | Example                | Description          |
| ---------- | ---------------------- | -------------------- |
| `normal`   | `line-height: normal;` | Default (~1.2)       |
| Numeric    | `line-height: 1.5;`    | Multiplies font size |
| Percentage | `line-height: 150%;`   | Based on font size   |
| Absolute   | `line-height: 24px;`   | Fixed spacing        |

---

## Example Usage

```css
.normal {
  line-height: normal;
}

.one-point-five {
  line-height: 1.5;
}

.double {
  line-height: 2;
}
```

### HTML Example

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    .normal {
      line-height: normal;
    }

    .one-point-five {
      line-height: 1.5;
    }

    .double {
      line-height: 2;
    }
  </style>
</head>
<body>

  <p class="normal">
    This is text with normal line height.
    This is text with normal line height.
  </p>

  <p class="one-point-five">
    This is text with line height 1.5.
    This is text with line height 1.5.
  </p>

  <p class="double">
    This is text with double line height.
    This is text with double line height.
  </p>

</body>
</html>
```

# 5.2 The `letter-spacing` Property

The `letter-spacing` property controls the spacing between individual characters.

## Why It Matters

* ✔ Improves readability
* ✔ Creates stylistic effects
* ✔ Useful for headings

---

## Values of `letter-spacing`

| Value    | Example                   | Description   |
| -------- | ------------------------- | ------------- |
| `normal` | `letter-spacing: normal;` | Default (0)   |
| Positive | `2px`                     | Adds space    |
| Negative | `-1px`                    | Reduces space |

---

## Example Usage

```css
.normal {
  letter-spacing: normal;
}

.wide {
  letter-spacing: 2px;
}

.narrow {
  letter-spacing: -1px;
}
```

### HTML Example

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    .normal {
      letter-spacing: normal;
    }

    .wide {
      letter-spacing: 2px;
    }

    .narrow {
      letter-spacing: -1px;
    }
  </style>
</head>
<body>

  <p class="normal">Normal letter spacing.</p>
  <p class="wide">Increased letter spacing.</p>
  <p class="narrow">Reduced letter spacing.</p>

</body>
</html>
```

# Best Practices for Text Spacing

* ✔ Use `line-height: 1.4–1.6` for readable paragraphs
* ✔ Avoid excessive negative spacing
* ✔ Use increased letter spacing for headings
* ✔ Keep body text spacing balanced
