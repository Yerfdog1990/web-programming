# Font Connection (Web Fonts)

Connecting fonts to a web page is essential for creating a unique and professional design. By default, browsers can only use fonts installed on the user’s computer. However, **web fonts** allow us to load fonts from a remote server so they can be displayed even if they are not installed on the user’s device.

---

# 3.1 Web Fonts

## What Are Web Fonts?

**Web fonts** are fonts that are loaded from a server and used on a webpage. This allows developers to use custom typography beyond system fonts.

### Why Use Web Fonts?

* Greater design flexibility
* Consistent appearance across devices
* Unique branding opportunities

---

## Connecting Web Fonts Using `@font-face`

The `@font-face` rule allows you to load custom font files from your own server.

---

### Step 1: Add Font Files to Your Project

Place font files inside a folder (for example, `fonts/`).

Common font formats:

| Format | Description                                     |
| ------ | ----------------------------------------------- |
| WOFF   | Web Open Font Format (widely supported)         |
| WOFF2  | Improved compression version of WOFF            |
| TTF    | TrueType Font                                   |
| EOT    | Embedded OpenType (for older Internet Explorer) |
| SVG    | Used in SVG graphics                            |

Recommended formats:

* `.woff2` (best compression)
* `.woff` (fallback support)

---

### Step 2: Use `@font-face` in CSS

```css
@font-face {
  font-family: 'MyCustomFont';
  src: url('fonts/MyCustomFont.woff2') format('woff2'),
       url('fonts/MyCustomFont.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}

body {
  font-family: 'MyCustomFont', sans-serif;
}
```

---

### Advantages of `@font-face`

* ✔ Full control over fonts
* ✔ Works without third-party services
* ✔ Compatible across browsers
* ✔ Supports multiple font formats

# 3.2 Google Fonts

Google Fonts is one of the most popular free web font services. It allows developers to easily integrate high-quality fonts.

---

## Steps to Connect Google Fonts

### 1. Choose a Font

Go to the Google Fonts website and select a font (e.g., Roboto).

### 2. Select Styles and Weights

Choose required font weights (e.g., 400, 700).

### 3. Copy the Generated Link

Google Fonts provides a `<link>` tag for your HTML file.

---

## Example Using Google Fonts (HTML Method)

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Google Fonts Example</title>

  <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

  <style>
    body {
      font-family: 'Roboto', sans-serif;
    }

    h1 {
      font-weight: 700;
    }
  </style>
</head>
<body>
  <p>This text is displayed using the Roboto font.</p>
</body>
</html>
```

---

## Why Use Google Fonts?

* ✔ Easy to integrate
* ✔ Large font collection
* ✔ Optimized delivery from fast servers
* ✔ Free to use

---

## Understanding `display=swap`

When web fonts load, there may be a delay before text appears.

`display=swap` improves performance by:

1. Showing system font immediately
2. Swapping to web font once loaded

This improves user experience, especially on slow internet connections.

---

# 3.3 Using CSS `@import`

Google Fonts can also be connected directly inside CSS using `@import`.

---

### Example:

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

body {
  font-family: 'Roboto', sans-serif;
}
```

### When to Use `@import`

* When working mainly inside CSS files
* When separating HTML and CSS strictly

However, the `<link>` method is generally preferred for better performance.

---

# Performance Considerations

Using too many fonts or font weights can slow down your website.

### Best Practices:

* ✔ Load only needed font weights
* ✔ Use `display=swap`
* ✔ Avoid too many font families
* ✔ Use caching and preload where possible

# Exercise 1: Font with `@font-face`

## Requirements

* Download **Open Sans** in `.woff2` and `.woff` formats
* Place them in a `fonts` folder
* Use `@font-face`
* Apply the font to:

    * `body`
    * `h1`

---

## ✅ Sample Solution

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    @font-face {
      font-family: 'OpenSansCustom';
      src: url('fonts/OpenSans.woff2') format('woff2'),
           url('fonts/OpenSans.woff') format('woff');
      font-weight: normal;
      font-style: normal;
    }

    body {
      font-family: 'OpenSansCustom', sans-serif;
    }

    h1 {
      font-family: 'OpenSansCustom', sans-serif;
    }
  </style>
</head>
<body>

  <h1>This heading uses Open Sans</h1>
  <p>This paragraph also uses Open Sans.</p>

</body>
</html>
```
