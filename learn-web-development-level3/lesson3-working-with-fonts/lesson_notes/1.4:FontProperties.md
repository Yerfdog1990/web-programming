# Font Properties in CSS

CSS provides several properties that allow you to control how fonts appear on a webpage. In this lesson, we focus on three important font properties:

* `font-style`
* `font-variant`
* `font-stretch`

These properties allow you to control text slant, capitalization style, and width.

---

# 4.1 The `font-style` Property

The `font-style` property defines whether text appears in a normal, italic, or oblique style.

## Values

| Value     | Description                                               |
| --------- | --------------------------------------------------------- |
| `normal`  | Regular text (default)                                    |
| `italic`  | Italic (cursive) text                                     |
| `oblique` | Slanted text (similar to italic but mechanically slanted) |

---

## Example Usage

```css
.normal {
  font-style: normal;
}

.italic {
  font-style: italic;
}

.oblique {
  font-style: oblique;
}
```

### HTML Example

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    .normal {
      font-style: normal;
    }

    .italic {
      font-style: italic;
    }

    .oblique {
      font-style: oblique;
    }
  </style>
</head>
<body>

  <p class="normal">This is normal text.</p>
  <p class="italic">This is italic text.</p>
  <p class="oblique">This is oblique text.</p>

</body>
</html>
```

---

## Difference Between Italic and Oblique

* **Italic** → Uses a specially designed italic font.
* **Oblique** → Slants the normal font artificially.

Not all fonts visually distinguish between italic and oblique.

---

# 4.2 The `font-variant` Property

The `font-variant` property controls how letters are displayed, especially in **small caps** format.

## Values

| Value        | Description                                             |
| ------------ | ------------------------------------------------------- |
| `normal`     | Regular text                                            |
| `small-caps` | Displays lowercase letters as smaller uppercase letters |

---

## Example Usage

```css
.normal {
  font-variant: normal;
}

.small-caps {
  font-variant: small-caps;
}
```

### HTML Example

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    .normal {
      font-variant: normal;
    }

    .small-caps {
      font-variant: small-caps;
    }
  </style>
</head>
<body>

  <p class="normal">This is normal text.</p>
  <p class="small-caps">This is text in small caps.</p>

</body>
</html>
```

---

## Additional Values (CSS Fonts Level 4)

More advanced values include:

* `all-small-caps`
* `petite-caps`
* `all-petite-caps`
* `unicase`
* `titling-caps`

⚠ These are not supported in all browsers.

---

# 4.3 The `font-stretch` Property

The `font-stretch` property controls the width of the font — making it more compressed or expanded.

It only works if the font family supports different stretch variations.

## Values

| Value             | Description     |
| ----------------- | --------------- |
| `normal`          | Default width   |
| `ultra-condensed` | Very narrow     |
| `extra-condensed` | Extra narrow    |
| `condensed`       | Narrow          |
| `semi-condensed`  | Slightly narrow |
| `semi-expanded`   | Slightly wide   |
| `expanded`        | Wide            |
| `extra-expanded`  | Extra wide      |
| `ultra-expanded`  | Very wide       |

---

## Example Usage

```css
.normal {
  font-stretch: normal;
}

.condensed {
  font-stretch: condensed;
}

.expanded {
  font-stretch: expanded;
}
```

---

### HTML Example

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    .normal {
      font-stretch: normal;
    }

    .condensed {
      font-stretch: condensed;
    }

    .expanded {
      font-stretch: expanded;
    }
  </style>
</head>
<body>

  <p class="normal">This is text with normal width.</p>
  <p class="condensed">This is condensed text.</p>
  <p class="expanded">This is expanded text.</p>

</body>
</html>
```
