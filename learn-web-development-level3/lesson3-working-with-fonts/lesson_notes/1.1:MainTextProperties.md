# Main Text Properties in CSS

CSS (Cascading Style Sheets) provides several properties to control how text appears on a webpage. The three **main text properties** are:

* `color` – controls text color
* `font-size` – controls text size
* `font-weight` – controls text thickness (light, normal, bold, etc.)

Understanding these properties allows you to design readable, attractive, and professional-looking web pages.

---

## 1. The `color` Property

The `color` property sets the color of text.

### ✅ Different Ways to Define Colors

CSS supports multiple color formats:

### 1. Color Names

```css
p {
  color: red;
}
```

### 2. Hexadecimal (Hex) Values

```css
p {
  color: #ff0000;
}
```

### 3. RGB and RGBA

```css
p {
  color: rgb(255, 0, 0);
  color: rgba(255, 0, 0, 0.5); /* Transparent red */
}
```

* `rgb()` defines red, green, blue values (0–255).
* `rgba()` adds transparency (alpha value from 0 to 1).

### 4. HSL and HSLA

```css
p {
  color: hsl(0, 100%, 50%);
  color: hsla(0, 100%, 50%, 0.5); /* Transparent red */
}
```

* `hsl()` stands for Hue, Saturation, Lightness.
* `hsla()` adds transparency.

---

### Example

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    p {
      color: #3498db; /* Blue color */
    }
  </style>
</head>
<body>
  <p>This text will be blue.</p>
</body>
</html>
```

---

## 2. The `font-size` Property

The `font-size` property controls how large or small the text appears.

### ✅ Common Units for `font-size`

### 1. Pixels (px)

```css
p {
  font-size: 16px;
}
```

* Fixed size.
* Most commonly used.

---

### 2. Relative Units

#### Percentage (%)

```css
p {
  font-size: 150%;
}
```

#### em

```css
p {
  font-size: 1.5em;
}
```

* Relative to the parent element’s font size.

#### rem

```css
p {
  font-size: 1.5rem;
}
```

* Relative to the `<html>` element’s font size.

---

### 3. Viewport Units

```css
p {
  font-size: 2vw; /* 2% of viewport width */
  font-size: 2vh; /* 2% of viewport height */
}
```

* `vw` = Viewport width
* `vh` = Viewport height
* Responsive to screen size

---

### Example

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    p {
      font-size: 18px;
    }
  </style>
</head>
<body>
  <p>This text will be 18 pixels.</p>
</body>
</html>
```

---

## 3. The `font-weight` Property

The `font-weight` property controls how thick or thin text appears.

### ✅ Keyword Values

```css
p {
  font-weight: bold;
}
```

Common keywords:

* `normal`
* `bold`
* `bolder`
* `lighter`

---

### ✅ Numeric Values

```css
p {
  font-weight: 300; /* Light */
  font-weight: 700; /* Bold */
}
```

| Value   | Meaning    |
| ------- | ---------- |
| 100–300 | Light      |
| 400     | Normal     |
| 700     | Bold       |
| 900     | Extra bold |

---

### Example

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    p {
      font-weight: 700;
    }
  </style>
</head>
<body>
  <p>This text will be bold.</p>
</body>
</html>
```

---

# Combined Example

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    p {
      color: #2c3e50;
      font-size: 20px;
      font-weight: 700;
    }
  </style>
</head>
<body>
  <p>This text is styled using color, font-size, and font-weight.</p>
</body>
</html>
```

# Exercise: Text Size and Text Weight

## Part 1: Text Size

### Requirements:

* One paragraph → 16px
* One paragraph → 1.5em
* One paragraph → 120%
* One paragraph → 2vw

---

## Part 2: Text Weight

### Requirements:

* One paragraph → `bold`
* One paragraph → `700`
* One paragraph → `300`
* One paragraph → `normal`

---

# ✅ Sample Solution

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    /* Text Size */
    .px {
      font-size: 16px;
    }

    .em {
      font-size: 1.5em;
    }

    .percent {
      font-size: 120%;
    }

    .vw {
      font-size: 2vw;
    }

    /* Text Weight */
    .bold {
      font-weight: bold;
    }

    .weight700 {
      font-weight: 700;
    }

    .weight300 {
      font-weight: 300;
    }

    .normal {
      font-weight: normal;
    }
  </style>
</head>
<body>

  <h2>Text Size Examples</h2>
  <p class="px">This paragraph is 16px.</p>
  <p class="em">This paragraph is 1.5em.</p>
  <p class="percent">This paragraph is 120%.</p>
  <p class="vw">This paragraph is 2vw.</p>

  <h2>Text Weight Examples</h2>
  <p class="bold">This paragraph is bold.</p>
  <p class="weight700">This paragraph has font-weight 700.</p>
  <p class="weight300">This paragraph has font-weight 300.</p>
  <p class="normal">This paragraph has normal weight.</p>

</body>
</html>
```

# Summary

The three main text properties in CSS are:

* `color` → Controls text color
* `font-size` → Controls text size
* `font-weight` → Controls text thickness

Mastering these properties allows you to:

* ✔ Improve readability
* ✔ Create visual hierarchy
* ✔ Design responsive layouts
* ✔ Control emphasis in your content

These are foundational CSS skills every front-end developer must understand.
